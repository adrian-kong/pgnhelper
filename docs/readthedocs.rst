Read The Docs
=============

**Requirements needed for building the docs.**

requirements.txt::

    Sphinx==5.0.2
    sphinxcontrib-napoleon==0.7
    sphinx-rtd-theme
    pgnhelper

Location: docs/requirements.txt

**Update docs**

command line::

    (venv) PS F:\Github\pgnhelper> cd docs
    (venv) PS F:\Github\pgnhelper\docs> make html
